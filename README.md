## Описание проекта

Решение направлено на оптимизацию профилактической работы в области надзора за эксплуатацией самоходных машин. На основе исторических данных (около 12 лет) строятся модели машинного обучения, которые рассчитывают индивидуальную вероятность нарушения обязательных требований для каждого владельца техники. Это позволяет проводить профилактические мероприятия **до** истечения срока технического осмотра, сокращая избыточные проверки и повышая точность выявления потенциальных нарушителей.

**Ключевые результаты:**
- Сокращение количества оформляемых предостережений на **60%** для законопослушных лиц.
- Охват профилактическими мероприятиями **85,4%** лиц, которые фактически допускают нарушение.

---

## Решаемая проблема

Ранее профилактика проводилась **по факту** нарушения, что противоречило реформе контрольно-надзорной деятельности (КНД). Инспекторам приходилось оформлять предостережения всем поднадзорным, включая добросовестных владельцев. Это создавало избыточную нагрузку и снижало доверие граждан.

Разработанное ИИ-решение позволяет:
- ежемесячно оценивать риски для ~11 000 лиц;
- выделять целевую группу с высоким риском (~3% от общего числа);
- проводить профилактику именно с теми, кто с высокой вероятностью нарушит сроки.

## Этапы обработки данных

1. **Загрузка данных** – чтение файлов `to.xlsx` и `tractors.xlsx`.
2. **Фильтрация актуальных записей** – исключение ликвидированных организаций и умерших лиц.
3. **Расчёт целевой переменной** – на основе исторических осмотров определяется факт нарушения сроков.
4. **Формирование признаков** – накопление статистики по владельцам и машинам по временной последовательности.
5. **Обучение модели** – настройка CatBoost с категориальными признаками.
6. **Прогнозирование** – расчёт вероятности нарушения для каждого владельца.
7. **Агрегация и отчёт** – консолидация результатов по владельцам, формирование Excel-файла.
8. **Сохранение** – вывод в `risk_june2026.xlsx`.
